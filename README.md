# 🧠 Memory Card Game

## 📌 Description
The Memory Card Game is an interactive web-based application designed to test and improve users' memory and concentration skills. The game challenges players to match pairs of cards within a limited time while tracking their scores and progress.

## 🎯 Objective
- To build an engaging and interactive game  
- To improve memory and concentration skills  
- To provide multiple difficulty levels  
- To store user progress and scores  

## 🚀 Key Features

### 🎮 Gameplay Features
- Card matching game with flipping animation  
- Three levels: Easy, Normal, Expert  
- Timer-based gameplay ⏱️  
- Score tracking system  
- Congratulatory messages on winning  

### 👤 User Features
- User Registration and Login  
- Personalized user profiles  
- Progress tracking  
- Score saving system  

### 🏆 Advanced Features
- Leaderboard system  
- Sound effects for actions  
- Dynamic UI with animations  
- Level unlocking system  

## 🛠️ Technologies Used
- HTML  
- CSS  
- JavaScript  
- PHP  
- MySQL  
- XAMPP (Local Server)  

## 🏗️ System Architecture
- Frontend: Handles UI and game interaction  
- Backend: Manages user authentication and data  
- Database: Stores user details, scores, and leaderboard  


## ⚙️ How to Run the Project

### 1. Clone Repository

### 2. Setup XAMPP
- Start Apache and MySQL  
- Place project folder inside `htdocs`  

### 3. Import Database
- Open phpMyAdmin  
- Create database  
- Import `.sql` file  

### 4. Run Project

http://localhost/project-folder-Name
